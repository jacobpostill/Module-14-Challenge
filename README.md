## Installation

To install this project download the repository onto your local drive. Then make sure you have node.js installed on your computer. Once the repository is cloned, open it, and in the command line run your mysql, then type "npm start" to start the generating process". Make sure you have the password for your sql in the server and prompt javascripts.
